# Rivia Magiarín

EN:
NFT art and storytelling website built with Flask and SASS.

Rivia Magiarín brings together three original collections—The Cozy Witch, Canary Feast, and Octopus—presented through an immersive visual experience with multilingual support (EN/ES/GL).
